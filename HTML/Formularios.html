Formularios en HTML

¿Qué son las etiquetas de formularios?

Los formularios son una de las herramientas principales para que los usuarios interactúen con una página web. Permiten ingresar datos, enviar mensajes, registrarse, iniciar sesión o seleccionar opciones dentro de un sitio.

En HTML, los formularios se construyen con la etiqueta <form> y dentro de ella se pueden usar diferentes campos como texto, correo, contraseña, archivos, fechas y casillas de selección.

Ejemplo de formulario

Sintaxis

<form>
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre">
</form>

Seleccionar archivo

El tipo file permite al usuario elegir un archivo desde su dispositivo.

Sintaxis

<input type="file">

Seleccionar fecha

El tipo date permite elegir una fecha desde un calendario.

Sintaxis

<input type="date">

Casilla de verificación

El tipo checkbox permite seleccionar o aceptar una opción.

Sintaxis

<input type="checkbox">
