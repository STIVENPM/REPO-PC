Texto en HTML

Jerarquía de títulos

En HTML la jerarquía de contenido se define utilizando las etiquetas de encabezado que van desde <h1> hasta <h6>. Cada número indica el nivel de importancia dentro del documento.

El <h1> representa el título principal de la página, mientras que <h2>, <h3> y los demás niveles se utilizan para dividir el contenido en secciones y subsecciones.

Es importante mantener un orden lógico de los encabezados para que los motores de búsqueda y los lectores de pantalla puedan entender correctamente la estructura del documento.

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Subsección</h3>

Formato de texto básico

El formato de texto en HTML permite organizar la información y dar significado al contenido. Algunas de las etiquetas más utilizadas son:

  • <p>: se utiliza para crear párrafos de texto.
  • <em>: se utiliza para dar énfasis a una palabra o frase, normalmente mostrándose en cursiva.
  • <br>: se utiliza para realizar un salto de línea dentro del texto.
<p>Este es un párrafo</p>
<em>Texto en cursiva</em>
Primera línea<br>Segunda línea

Enlaces en HTML

Los enlaces se crean con la etiqueta <a> y permiten conectar páginas web entre sí mediante el atributo href.

Enlace externo

Un enlace externo dirige al usuario hacia otra página web diferente.

Ir a Google

Sintaxis

<a href="https://www.google.com">Ir a Google</a>

Enlace interno

Un enlace interno permite navegar entre páginas dentro del mismo sitio web.

Ir a Inicio

Sintaxis

<a href="index.html">Ir a Inicio</a>

Insertar un sitio web

La etiqueta <iframe> permite mostrar otra página web dentro de la página actual.

Sintaxis

<iframe src="https://www.wikipedia.org"></iframe>
