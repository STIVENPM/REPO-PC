Listas en HTML

Las listas en HTML permiten organizar información de forma clara y estructurada. Se utilizan para mostrar elementos relacionados dentro de una misma categoría y facilitan la lectura del contenido.

Existen principalmente dos tipos de listas en HTML:

Listas ordenadas

Las listas ordenadas utilizan la etiqueta <ol> y muestran los elementos en un orden específico utilizando números o letras.

Sintaxis

<ol>
  <li>Abrir el editor de código</li>
  <li>Crear un archivo HTML</li>
</ol>

Ejemplo

  1. Abrir el editor de código
  2. Crear un archivo HTML
  3. Escribir la estructura básica

Listas desordenadas

Las listas desordenadas utilizan la etiqueta <ul> y muestran los elementos mediante viñetas. Se utilizan cuando el orden de los elementos no es importante.

Cada elemento dentro de una lista se define con la etiqueta <li> (list item), que representa cada elemento de la lista.

Sintaxis

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Ejemplo

  • HTML
  • CSS
  • JavaScript
